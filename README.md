## 👨‍💻 Sobre Mim

Sou estudante de Engenharia da Computação na FIAP, apaixonado por tecnologia e desenvolvimento de software. Meus principais interesses orbitam em torno da arquitetura de software, desenvolvimento web e o entendimento aprofundado de sistemas operacionais. Utilizo o Linux Mint como meu ambiente de trabalho principal, onde exploro desde diagnósticos de hardware e configurações de kernel até projetos desafiadores envolvendo visão computacional.

No campo prático, venho consolidando minha base técnica em diferentes linguagens e ecossistemas. Desenvolvo projetos com foco em automação, inteligência artificial aplicada e arquitetura robusta, construindo desde interfaces web modernas até aplicações desktop de alta fidelidade.

Fora do universo do código e dos estudos acadêmicos, dedico meu tempo livre à leitura e a jogos de estratégia. Sou motivado pela oportunidade de solucionar problemas reais e construir soluções eficientes através do desenvolvimento de software.

## 🚀 Projetos em Destaque

* **ProSec:** Um sistema de monitoramento em tempo real operado na borda (*Edge Computing*) projetado para segurança proativa e preditiva. A stack do projeto utiliza YOLOv8 Nano, YOLO Pose Estimation, Scikit-Learn / XGBoost, PyQt e SQLite3 / PostgreSQL.
* **Reservas FIAP:** Uma aplicação desktop robusta desenvolvida em Python e integrada com SQLite3, projetada para a gestão otimizada de agendamentos acadêmicos.
* **Site Ecommerce:** Uma interface de e-commerce de alta fidelidade, desenvolvida como um desafio front-end utilizando React e TypeScript.
* **Analisador HTML (Java):** Ferramenta focada em análise de código, com documentação detalhada e *showcase* no meu portfólio.
